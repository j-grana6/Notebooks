Note 1: Strategies that give the same on-equilibrium path observations do not necessarily have the same intelligence¶

Consider player 1's strategy:

  • $A$ if 1, $B$ if 2

and the following three profiles by player 2:

  • $X$ if $A$, $Y$ if $B$, $X$ if $C$
  • $X$ if $A$, $Y$ if $B$, $Y$ if $C$
  • $X$ if $A$, $Y$ if $B$, $Z$ if $C$

Of course, these three strategies will give the same observed outcome. In fact, using the first and the third profile for player 2 are both Nash equilibria. (They are the "good" equilibria)

Do why doesn't the second strategy of player 2 give an intelligence of 1? The reason is that if player 2 plays $Y$ when player $1$ plays $C$, then player one will have a higher utility if his strategy has him playing $C$ when the market is state 2.

Note 2¶

"Not good" equilibria can be in an equivalence class with other profiles that do not have an intelligence of 1. For example, consider the equivalence class that has player 1 playing $A$ when nature is in state $1$ and $C$ when nature is in state 2. Player 2 plays $X$ when he observes $A$ and $Z$ wgen he observes $C$. Of course there are 3 strategies in this equivalence class. However, two of these strategies do not have intelligence of 1 because player 2 no longer makes a "threat" and therefore player 1 can earn a higher utility by altering his strategy.

All Intelligences¶

To read this, the first tuple is player 1's strategy. The first element is what he would do if the market is in state $1$, the second is what he would do in state 2. The second tuple is player 2's strategy. Each element corresponds to what he observes from "A'. For example, the 4th strategy profile is:

Player 1 plays $A$ in market state 1, player 1 plays $A$ in market state 2. Player 2 plays $X$ if he observes $A$, $Y$ if he observes $B$ and $Z$ if he observes $C$.

"Full Game" Intelligence¶

Here we combine the real intelligence with the full game intelligence described by James as:

  1. Calculate the intelligence of s_L and s_F in the usual way, i.e. on-path, and call it IQ(i,s,on-path). While doing that, keep track of all unvisited information sets for each player, i.e. the set of off-path information sets, H_i.

  2. For each player i, for each unvisited information set h in H_i, and for each state n in h, calculate the intelligence of s_i in information set h given that the state is n, call this IQ(i,s,n).

  3. For each player i, for each unvisited information set h, take the average intelligence across nodes n in h: IQ(i,s,h)=1/|h| * \sum_{n\in h} IQ(i,s,n)

  4. Take the average of IQ(i,s,h) across unvisited information sets in Hi: IQ(i,s,off-path) = 1/|H_i| * sum{h\in H_i} IQ(i,s,h).

  5. Calculate IQ(i,s,full-game) = pIQ(i,s,on-path) + (1-p)IQ(i,s,off-path), where p is a parameter we control based on the importance of off-path play. Let’s start with something like p=.5.

def get_off_path_intel(profile):
    unvisited = list(get_off_path_info_sets(profile))
    unvisited_intels = []
    for u in unvisited:
        num_no_better_high = 0.
        num_no_better_low = 0.
        p2s_action = profile[1][mapdict[u]]
        utility_high = utils["H" + u + p2s_action][1]
        utility_low = utils["L" + u + p2s_action][1]
        p2s_other_actions = list(set(["X", "Y", "Z"]) - set(p2s_action))
        for other in p2s_other_actions:
            utility_high_other = utils["H" + u + other][1]
            utility_low_other = utils["L" + u + other][1]
            if utility_high_other <= utility_high:
                num_no_better_high +=1
            if utility_low_other <= utility_low:
                num_no_better_low +=1
        unvisited_intels.append(.5 * num_no_better_high/2. + .5* num_no_better_low/2.)

        
    return sum(unvisited_intels)/float(len(unvisited_intels))

Note 1¶

The "good" equilibria do not have off-path intellifences of 1.
The reason is that the intellegience is the average of the intelligence that are computed by holding the state constant. Therefore, if the optimal action in an off path information set is different for at least two of the states, the off path intelligence will be less than 1.
