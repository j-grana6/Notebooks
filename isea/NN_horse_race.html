Neural Network Horse Race¶

Initial Meanderings¶

The goal of this project is to forecast annual zip code level data. In fact, there are two problems tangled into one. The first is to optimally interpolate monthly zip code level data from annual zip code level data and monthly metropolitan level data. The second task is to forecast zip code level data into the future (and the past). To better understand the problem, consider one metropolitan statistical association ($MSA$) and the zipcodes ($Z$) that belong to that $MSA$. The variables $MSA$ and $Z$ represent some variable of interest (i.e. employment) for the metropolitan as a whole and each zipcode, respectively Initially, we will assume that each $MSA$ is independent so that we only consider one $MSA$ in the initial exploration.

  • $MSA$ is a scalar variable and is indexed by time, $t$.
  • $Z$ is a vector variable that contains data for each zip-code.
  • $Z_t$ is generated by an (stochastic) autoregressive process, $Z_t =g(Z_{t-1}, \; Z_{t-2}...)$
  • There exists some (possibly stochastic) function $h$ such that $MSA_t=h(Z_t)$. Intuitively, $h$ is the function that aggregates zip-code level data to the metropolitan level. The stochasticity can be attributed to either imperfect data or simple statistical discrepancy that results from different data collection procedures. The inverse of $h$ would simply be an de-aggregator function.

Simple Examples¶

Suppose the $Z_t$ follows an AR1 process, and each $z_t$ within $Z_t$ are independent (no spatial spillovers), Then we can write \begin{equation} Z_t = \mathbf{\rho}Z_{t-1}' + \mathbf{\epsilon} \end{equation}

where $Z$ and $\mathbf{\rho}$ are $|Z|\times 1$ vectors and ' denotes vector transpose. In this simple scenario, we can estimate $\rho_i$ separately for each element of $z_i$ in $Z$ and from this, the optimal forecast is straight forward. This is unsatisfactory for the following reasons:

1) There are likely to be spatial dependencies among zip-codes.

2) If we let the frequency of $t$ be months and we have data for $Z_t$, then we do not have data for $Z_{t-1}$ but only have data for $Z_{t-12}$.

3) There is information in the $MSA_{t-1}$ variable that might help us to predict the value of $Z_t$.

One solution to addressing issue 1 is to change $\mathbf{\rho}$ from a vector to a $|Z| \times |Z|$ matrix. Then we can do a VAR to get the optimal forecast.

However, the more relevant and interesting issue is to determine how to incorporate the data from the $MSA$ variable into the optimal forecast.

Assuming a simple aggregation function¶

One way to incorporate information in the MSA variable is to carry out a simple procedure:

1) Assume a functional form for the zipcode data

2) Select and aggregation function and derive its closed form from the zipcode data

3) Forecast $MSA_{t+k}.$

4) Forecast $Z_{t+k} = h^{-1}(MSA_{t+k})$

This basically turns out to be a two step estimation process. First, we estimate the parameters of the zipcode level function. Using this we forecast MSA nd then de-aggregate.

Working through an example¶

Suppose:

1) We have two zip-codes, $z_1$, $z_2$, in one MSA.

2) The aggregating function is loosely defined by the equations $z_{1,t} + z_{2,t} = MSA_t$ Intuitively, the two zipcodes exactly determine the MSA.

3) Assume $Z_t$ is a multivariate AR(1) process with 0 correlation. That is $z_{1,t} = \mu_1 + \rho_1 z_{1, t-1} + \epsilon_1$ and $z_{2,t} = \mu_2 + \rho_2 z_{2, t-1} + \epsilon_2$ where the $\epsilon$ terms are independent.

4) The aggregation function is $z_{1_t} + z_{2,t} = MSA_t$. That is, there is no statistical discrepency between the zipcode level data and the metro level data.

5) This that the DGP for $MSA_t$ is

\begin{equation} MSA_t = \mu_2 + \mu_1 + \rho_1 z_{1, t-1}+ \rho_2 z_{2, t-1} + \epsilon_1 + \epsilon_2 \end{equation}

6) With a bit of hand waving, we can show that the distributions of $z_{1,t}$ and $z_{2,t}$ given $MSA_t$ are normal and derive the mean and the variance. We can also show that $MSA_t$ follows (at most) an AR(2) process.

Forecast procedure:¶

1) Estimate the process governing $Z_t$ so that we can derive the conditional distirbution of $Z_t$ given $MSA_t$.

2) Estimate the autoregressive process governing $MSA_t$.

3) Forecast $MSA_t$ to the period of interest

4) Use the conditional distribution of $Z_t$ given $MSA_t$ to forecast $Z_t$.
