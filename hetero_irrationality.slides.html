Overview¶

  • Order book market
  • Agents make valuation of underlying asset
  • Three types of agents: institutions, conservative traders, trend followers
  • Decide to place order depending on how much the observed price differs from their individual valuation

Valuations¶

Institutions¶

\(P_{i,t} = P^*_t + \delta_{i,t}\)

Where \(P_{i,t}\) is the institutions valuation, \(P^*_t\) is the fundamental value and \(\delta_{i,t}\) is a mean 0 normal random variable with \(\sigma \geq 0\)

Conservative Trader¶

\(P_{c,t} = \beta_cP_{t-1} + (1-\beta_c)P_{c, t-1} +\delta_{c,t}\)

where \(\beta\) is a smoothing parameter and \(P_{t-1}\) is the observed price last period.

\(\beta\) can be thought of as an parameter of belief stickiness

Trend follower¶

\(P_{f, t} = \hat{\pi_0} + \hat{\pi_1}(h)\)

where the \(\hat{\pi}\)'s are obstained through linear regression on the past \(h\) prices.

\(h\) can be interpreted as a horizon

Participation probability¶

Note that \(d_i\) is the relative deviation of the observed price from the agent's valuation. Then:

\(f_i(d_{i,t}) = \int_0^{d_{i,t}} \phi_i e^{-\phi_i d_{i,t}} = 1 - e^{-\phi_id_{i,t}}\)

\(\phi\) is loosely a risk parameter

Order Price¶

\(P(ticks away = x) =\Omega \big[\int_0^{x+1}\lambda_{i,t}e^{-\lambda_{i,t}(x+1)}-\int_0^x\lambda_{i,t}e^{-\lambda_{i,t}(x+1)}\big]= \Omega \big[e^{-\lambda_{i,t}(x)} -e^{-\lambda_{i,t}(x+1)}\big]\)

Where \(\lambda_{i,t} = d_{i,t} / \rho_i.\)

  • Agents can only order in $0.10 intervals
  • Limit order must be within $2.00 of observed price.

\(\rho\) can be interpreted as a patience parameter

Order Quantity¶

\(q_{i,t} = \mathcal{N}(\mu_i, \Sigma_i) + \psi_i \Sigma_Id_{i,t}\)

\(\mu_i\) and \(\Sigma_i\) can be interpreted as trader size parameters

\(\psi\) can be interpreted as a reaction parameter

Summary¶

  • Initialize order book with standing orders
  • Initialize starting price of the stock to 100
  • for day in NumDays
    • Fundamental value moves
    • for tick in NumAgents
      • Sample agent without replacement
      • Agent forms valuation of stock \((\beta, h, \delta)\)
      • Agent calculates relative difference between valuation and stock price
      • Agent decides if the potential order side is buy or sell
      • If agent has a limit order that contradicts his valuation: cancel order
      • Agent decides whether or not to enter market \((\phi)\)
      • If agent does not enter market: END tick
      • Agent cancels outstanding limit orders
      • Agent decides order price \((\rho)\)
      • Agent decides order quantity \((\mu, \Sigma, \psi)\)
      • Market order executed or limit order placed in book
